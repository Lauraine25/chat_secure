🚀 INSTALLATION
═════════════════════════════════════════════════════════════

TEMPS TOTAL: ~15 minutes

Étape 1: Prérequis
  • Python 3.8+
  • MySQL Server (via XAMPP)
  • ~200 MB disque libre

Étape 2: Cloner/Télécharger
  • Copier dans c:\xampp\htdocs\chat_secure\
  • Ou cd à votre répertoire

Étape 3: Environnement
  • python -m venv venv
  • .\venv\Scripts\activate (Windows)
  • source venv/bin/activate (Linux/Mac)

Étape 4: Dépendances
  • pip install -r requirements.txt
  • ~30 secondes

Étape 5: Base de Données
  • python setup.py (AUTOMATIQUE)
  • Ou: mysql -u root < database_setup.sql

Étape 6: Lancer
  • python server.py (Terminal 1)
  • python client.py (Terminal 2+)


▶️ UTILISATION
═════════════════════════════════════════════════════════════

CLIENT:
1. Lancer: python client.py
2. Entrer username (ex: alice)
3. Cliquer "Connecter"
4. Sélectionner destinataire
5. Écrire message
6. Cliquer "Envoyer"
→ Message chiffré automatiquement!

SERVEUR:
1. Lancer: python server.py
2. Voir: "Serveur démarré sur localhost:5555"
3. Chaque connection: log de connexion
4. Chaque message: log + sauvegarde BD
